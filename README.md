# ConstructionOS

ConstructionOS is the initial SaaS demo for construction companies, architecture studios, and site coordinators.

This first local scaffold focuses on a public landing page, lead capture, a base dashboard, and an initial opportunities CRM module prepared for Supabase.

## Local Commands

```bash
npm run dev
npm run lint
npm run build
```

## Environment

Copy `.env.example` to `.env.local` locally when Supabase credentials are available. Do not commit `.env.local`.
